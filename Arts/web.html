Web Design

Štampirana

In the Advanced Web Design course, I had a project to create a website for a company of my choice. It was necessary to use JavaScript, in addition to HTML and CSS, to improve the UX.
The website had to be responsive, so that the project content was displayed consistently on the phone, tablet, and desktop. I chose the company - Štampirana.
Štampirana is a 3D printing company that was founded by three brothers to help architecture students to print their models at affordable prices.

Developed using: HTML5, CSS3, JavaScript
Year: 2021.
